# 茶百道经营新引擎 · 微企付微资金方案

> 微企付 × 茶百道 · 可交互演示原型
>
> 国有银行账户体系 + 腾讯混元大模型 3.0，把全国 6,800+ 门店的钱、单、货串成一笔账。

## 演示内容（7 屏）

| # | 屏 | 看点 |
|---|---|---|
| 01 | 封面 Hero | 80px 大字标题 + 4 个核心指标 |
| 02 | 痛点 | 钱散 / 账乱 / 货误 三个连锁餐饮的真问题 |
| 03 | 整体框架 | L1–L5 六层结构总图 |
| 04 | 资金归集 | **可交互**：勾选 5 大渠道，账户余额实时联动 |
| 05 | 应用闭环 | 提现 / 订货 / 调拨 三个出口 |
| 06 | AI 三能力 | **核心可交互**：店长 / 老板 / 总部 三角色 Tab |
| 07 | 差异化 | 与银行账户产品 8 维对比 + 金句 |

### 屏 06 交互亮点

**Tab A · 门店店长视角**
- 5 个 SKU 库存条可点击调整水位
- 右侧 AI 推送卡片实时联动（标题、ETA、补货清单、金额）
- 一键下单按钮反馈

**Tab B · 多店老板视角**
- 5 店对账表，每行可勾选/取消
- 底部 4 个汇总指标实时更新
- 重新对账按钮 loading 动画
- 智能调拨卡片点击完成调拨

**Tab C · 茶百道总部视角**
- 四组筛选器（区域/能级/地段/品类）
- 华东热力地图 + 4 条 AI 文字洞察
- 4 个总部级 KPI

## 设计风格

- 主色：微信绿 `#07C160` + 深绿 `#0E2D1F`
- AI 高亮：橙色 `#FA9D3B`
- 字体：TencentSans + Source Han Serif（衬线引言）+ SF Mono（数字）
- 视觉：白底 + 微弱 mist 渐变 + 大宋体水印汉字
